Changee Learning Path - 與 API 合作

在網路的時代，我們在開始學習新領域時，可以透過 Google 找到許多學習的「資訊」，然而過多的資訊反而無法判斷什麼才是適合自己的「資源」。現有的學習資源非常豐富，但卻散落在各處，使得「自我學習」時總是花費大量時間在搜尋適合自己的資訊。因此我們希望能夠建立 Changee Learning Path「學習路徑平台」，整合資源與資訊，透過「學習分類帽」、「學習軌跡」、「實體資源」三個主要項目，建立方便自學的「學習路徑平台」。

運算子

  • 加
    6 + 4 → 10
  • 減
    9 - 5 → 4
  • 乘
    4 * 3 → 12
  • 除
    12 / 4 → 3
  • 餘數
    43 % 10 → 3
  • 大於
    6 > 4 → true
  • 小於
    9 < 5 → false
  • 大於等於
    8 >= -2 → true
  • 等於
    3 == 4 → false
  • 不等於
    12 != 4 → true

變數 var

var learningPath = 3;
  • var - 變數
  • learningPath - 變數名稱
  • ＝ - 代入運算子
  • 3 - 變數值

變數命名方式

    不好的示範 
  • var no space; // 空格
  • var 3blindmice; // 數字開頭
  • var scored_is_fine; // 用底線分隔字
  • var $_$; // 無意義的字
  • var mortalKombat2; // 使用數字當序號
    建議使用 
  • var goodNameBro; // 開頭小寫，後面開頭大寫（馱峰命名）

物件 Object、屬性 Property

  • 建構式（不建議使用）
    var item = new Object();
  • 物件實字（建議使用）
    var item = {};
  • 屬性
    var mark = {
  career: 'Front-End Developer',
  age: 27
};
  • career, age 都是物件 mark 的屬性
    mark.age → 27
mark['career'] → 'Front-End Developer'

陣列 Array

  • 建構式（不建議使用）
    var items = new Array();
  • 物件實字（建議使用）
    var items = [];
  • 將值放入陣列
    var items = [];
items.push('MacBook Air');
items.push('iPhone5s');
console.log(items); → ['MacBook Air','iPhone5s']
  • 讀取陣列中的值
    items[0] → 'MacBook Air'

用 Firebase 快速完成社群登入

Firebase

  • Real-Time 即時
  • 已經寫好 Simple Login
  • 等於是一個後端

Facebook 登入步驟

  1. 申請 facebook App 取得 AppID 與 secret
  2. 設定 website URL 為 https://auth.firebase.com/auth/facebook/callback
  3. 在 Firebase 裡設定 url, AppID, secret
  4. 引入 Firebase Javascript SDK

登入程式碼

var ref = new Firebase("https://{YOUR-FIREBASE}.firebaseio.com");
var authClient = new FirebaseSimpleLogin(ref, function(error, user) {
  if (error !== null) {
    // 使用者拒絕授權
    console.log("Error authenticating:", error);
  } else if (user !== null) {
    // 使用者成功登入
    console.log("User is logged in:", user);
  } else {
    // 使用者登出
    console.log("User is logged out");
  }
});

登入執行

auth.login("facebook", {
  scope: "public_profile, user_friends" // facebook Permission.
});
